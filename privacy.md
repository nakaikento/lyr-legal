## 1. はじめに

本プライバシーポリシーは、LYR (以下「本サービス」) が、ユーザーの情報をどのように取り扱うかを定めるものです。本サービスを利用することで、ユーザーは本ポリシーに同意したものとみなされます。

---

## 2. 取得する情報と利用目的

本サービスは Android 端末上で動作する画面翻訳アプリです。以下の情報のみを処理します。

### 2.1 画面キャプチャ

- **目的**: OCR（文字認識）による翻訳対象テキストの抽出
- **処理方法**: ユーザーが翻訳を実行した瞬間に Android の MediaProjection API でスクリーンショットを取得し、**端末内のみ**で処理します
- **外部送信**: 画像そのものは一切外部へ送信しません

### 2.2 OCR で抽出したテキスト

- **目的**: 翻訳 API への入力
- **処理方法**: 画面から抽出したテキストを、米国所在の大規模言語モデル (LLM) を利用した翻訳 API サービスへ HTTPS 経由で送信します
- **送信範囲**: 翻訳に必要なテキストのみ。ユーザー識別子や位置情報などは送信しません

### 2.3 ローカル設定

以下は端末内の DataStore / SharedPreferences にのみ保存され、外部送信されません。

- 翻訳元・翻訳先言語設定
- テーマ設定（ライト / ダーク）
- 翻訳回数カウンタ（日次リセット）
- 共有先アプリのパッケージ名（ユーザーが選択したアプリのみ）
- トークン使用量・課金プラン状態

---

## 3. 第三者への情報提供 / 国外移転

### 3.1 翻訳 API サービス

本サービスは翻訳処理を **米国所在の大規模言語モデル (LLM) 翻訳 API サービスプロバイダー** に委託しています。OCR で抽出されたテキストは HTTPS で当該プロバイダーへ送信されます。

- **移転先国**: アメリカ合衆国
- **移転する情報**: 画面から OCR で抽出された翻訳対象テキスト
- **移転の目的**: 翻訳結果の取得のみ
- **ユーザー識別子**: 送信しません
- **プロバイダー名の個別開示**: ご希望の方は本ポリシー末尾の連絡先までお問い合わせください

利用中のプロバイダーは、品質・速度・コスト等の観点から予告なく変更される場合があります。重要な変更があった場合は本ポリシーの「最終更新日」を更新して周知します。

### 3.2 その他

上記以外の第三者へは情報を提供しません。アクセス解析・広告ネットワーク等は導入していません。

---

## 4. 情報の保存期間

- **画面キャプチャ**: 翻訳処理終了後、即座にメモリから破棄
- **OCR テキスト / 翻訳結果**: 端末内の翻訳キャッシュ（MD5 ハッシュ基準）に保存し、ユーザーがアプリを削除すると同時に完全削除
- **サーバー側の保存**: 本サービスは独自サーバーを持たず、ユーザーデータを一切保存しません

---

## 5. 権限の利用

本サービスは以下の権限を利用します。

| 権限 | 用途 |
|------|------|
| `SYSTEM_ALERT_WINDOW` | 翻訳結果のオーバーレイ表示 |
| `FOREGROUND_SERVICE` / `FOREGROUND_SERVICE_MEDIA_PROJECTION` | 画面キャプチャの常駐実行 |
| `POST_NOTIFICATIONS` | Foreground Service 通知の表示 |
| `VIBRATE` | 操作フィードバック（触覚） |
| `INTERNET` | 翻訳 API への翻訳リクエスト送信 |
| `QUERY_ALL_PACKAGES` | 共有先アプリの表示名取得 |
| Accessibility Service | スクロール停止検知・投機的 OCR トリガー（画面内容の読み取りは行いません） |

---

## 6. Accessibility Service の利用について

本サービスは Accessibility Service を「スクロール停止の検知」と「フォアグラウンドアプリのパッケージ名取得」にのみ利用します。`canRetrieveWindowContent = false` 相当の最小情報のみを取得し、画面内のテキスト内容やユーザー入力は読み取りません。

---

## 7. 年齢制限

本サービスは **13 歳未満** の利用を想定していません。13 歳未満の方は、保護者の同意のもと利用するか、利用を控えてください。

---

## 8. データの削除・エクスポート

すべてのユーザーデータは端末内にのみ保存されます。アプリをアンインストールすれば、すべてのローカルデータが削除されます。サーバー側には何も保存していないため、別途のデータ削除依頼は不要です。

---

## 9. ポリシーの変更

本ポリシーは必要に応じて改訂されます。重要な変更がある場合は、アプリ内告知または本ポリシー冒頭の「最終更新日」更新で周知します。
